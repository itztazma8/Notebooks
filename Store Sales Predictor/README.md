## Store Sales Predictor
This is a project on sales prediction based on different types of stores. This division is done based on area. There are a few other things that have been considered. 
<br><br>
**Dataset**: [Stores](https://www.kaggle.com/datasets/surajjha101/stores-area-and-sales-data/data)

#### Category
There are three categories- *Small*, *Medium*, *Large*
#### Customer Preference
The number of customers with respect to the area of the stores has been calculated
#### Days Of Stock
The number of products that can be stocked even after doing the sales in a month is included
#### Sales Index
Sales Index is the sales that a shop made, considering the maximum sales made by a shop in that category
